# oobb_object_bearing

**OOBB Object: Bearing**

Generates a bearing model (for example 606/608 series) for rotational assemblies.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| bearing_name | Bearing designation (e.g. 606, 608). | string | 608 |
| exclude_clearance | Exclude outer clearance geometry if true. | boolean | False |
