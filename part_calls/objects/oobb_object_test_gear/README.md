# oobb_object_test_gear

**OOBB Object: Test Gear**

Generates a parametric gear test plate exploring pressure_angle, clearance, and backlash variations.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| style | Gear style variant. | string | socket_cap |
| pos | Position offset [x, y, z]. | array | [0, 0, 0] |
| full_object | Whether to return full thing dict or just components. | boolean | True |
