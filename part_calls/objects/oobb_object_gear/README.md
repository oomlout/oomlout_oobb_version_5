# oobb_object_gear

**OOBB Object: Gear**

Generates parametric OOBB-compatible gears for rotational assemblies.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| teeth | Number of gear teeth. | number | 20 |
| thickness | Gear thickness in mm. | number | 3 |
| size | Grid system (oobb/oobe). | string | oobb |

## Category

OOBB Geometry
