# oobb_object_holder

**OOBB Object: Holder**

Generates holder-style OOBB geometry for mounting electronic or mechanical components.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| width | Holder width in OOBB grid units. | number | 3 |
| height | Holder height in OOBB grid units. | number | 3 |
| thickness | Material thickness in mm. | number | 3 |
| size | Grid system (oobb/oobe). | string | oobb |

## Category

OOBB Geometry
