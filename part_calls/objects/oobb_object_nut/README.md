# oobb_object_nut

**OOBB Object: Nut**

Generates a hex nut matching the requested hardware thread size.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| radius_name | Thread size designation (e.g. m3, m5). | string | m3 |
| depth | Nut thickness/depth in mm. | number | 3 |
